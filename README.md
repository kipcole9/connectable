# Connectable

Simplify many-to-many connections for ActiveRecord 

Example:

```ruby
class Thing < ActiveRecord::Base
  connects_to :image

end
```

## Installation

Add this line to your application's Gemfile:

    gem 'connectable'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install connectable
